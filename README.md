# 🍭 Nassau Candy Sales Dashboard

## 📊 Project Overview
This project is a Sales and Profitability Analysis Dashboard built using Streamlit.  
It helps analyze sales, profit, and margin using interactive charts and filters.

## 🚀 Features
- Product profitability analysis
- Division performance comparison
- Cost vs Sales analysis
- Pareto analysis (80/20 rule)
- Profit prediction using Linear Regression
- Filters: Date, Division, Margin, Product search

## 🛠️ Tools Used
- Python
- Pandas, NumPy
- Plotly
- Streamlit
- Scikit-learn

## 📈 Key Insights
- Few products generate most of the profit
- Strong relationship between sales and cost
- Some products have low margin (high risk)

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py
