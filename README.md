# 🍭 Nassau Candy Dashboard

## 📊 Project Overview

This project is a **Sales and Profitability Analysis Dashboard** built using Streamlit.
It helps analyze business data such as sales, cost, profit, and margin using interactive filters and visualizations.

---

## 🚀 Features

* Product-level profit analysis
* Division-wise performance comparison
* Cost vs Sales visualization
* Margin distribution analysis
* Pareto analysis (Top contributing products)
* Profit prediction using Linear Regression
* Filters:

  * Date range
  * Division
  * Margin threshold
  * Product search

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Streamlit
* Scikit-learn

---

## 📈 Insights

* A small number of products generate most of the profit
* There is a strong relationship between sales and cost
* Some products have low margin and may be high risk

## ▶️ How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/M1812201/my-candy-project.git
   ```

2. Navigate to the project folder:

   ```
   cd my-candy-project
   ```

3. Install required libraries:

   ```
   pip install -r requirements.txt
   ```

4. Run the application:

   ```
   streamlit run app.py
   ```
